Broken Links Checker is a WordPress plugin that helps you find and fix broken links on your website.

== Description ==

Broken links can negatively impact your website's SEO and user experience. Broken Links Checker scans your website for broken links and provides a report so you can easily fix them.

== Features ==

* Scans your entire website for broken links
* Provides a report of all broken links
* Allows you to export the report in CSV format

== Installation ==

1. Upload the `wpblc-broken-links-checker` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Broken Links Checker page in the admin menu to start scanning your website for broken links

== Usage ==

1. Go to the Broken Links Checker page in the admin menu
2. Click on the 'Scan' tab
3. Click on the 'Export Report in CSV' button to export the report in CSV format

== Frequently Asked Questions ==

= Does this plugin scan all links on my website? =

Yes, Broken Links Checker scans all links on your website, including links in posts, pages, and comments.

= How often should I scan my website for broken links? =

It's recommended to scan your website for broken links at least once a month. However, if your website has a lot of content or if you update your website frequently, you might want to scan your website for broken links more often.
